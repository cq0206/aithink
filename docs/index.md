## 今日必读

### [Anthropic 多智能体协作指南【译】导读与点评](/sources/dotey-2043240706156728322)

为什么推荐：这篇最有价值的地方，不是列举了五种模式，而是强调“先从最简单、最能跑通的模式开始”，这比一上来堆复杂架构更像成熟工程判断。

### [Claude Code 专题](/sources/claude-code)

为什么推荐：如果你最近在关注 coding agent、harness engineering 或 Claude Code，这个专题已经具备比较完整的阅读路径，适合连续阅读而不是零碎跳转。

### [OpenAI：Designing delightful frontends with GPT-5.4](/sources/openai-gpt54-frontends)

为什么推荐：这篇适合前端和产品团队，因为它讲的不是抽象 AI 概念，而是怎样把模型能力落到真实界面和交互质量上。

## 这个站怎么用

1. 先看“今日必读”，快速决定今天最值得花时间的 1 到 3 篇。
2. 再看 [AI Think Brief](/brief/)，跟上本周最值得读的主线。
3. 如果某个主题值得深挖，进入对应专题连续阅读。
4. 最后用 [90 天路线](/learning/roadmap) 把阅读变成长期学习系统。

## 编辑方法

这个站的内容不是“看到就收”，而是按下面的顺序筛选：

1. 只收录高信号来源
2. 优先保留有长期价值的文章，而不是只追热点
3. 尽量为重点内容补充“为什么推荐”
4. 对适合长期沉淀的主题做专题化整理

如果你第一次来，建议先从 [AI Think Brief](/brief/) 和 [Claude Code 专题](/sources/claude-code) 开始。
