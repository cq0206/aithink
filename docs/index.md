## 这个站怎么用

1. 先看“今日必读”，快速决定今天最值得花时间的 1 到 3 篇。
2. 再看 [AI Think Brief](/brief/)，跟上本周最值得读的主线。
3. 如果某个主题值得深挖，进入对应专题连续阅读。
4. 最后用 [90 天路线](/learning/roadmap) 把阅读变成长期学习系统。

## 编辑方法

这个站的内容不是“看到就收”，而是按下面的顺序筛选：

1. 只收录高信号来源
2. 优先保留有长期价值的文章，而不是只追热点
3. 尽量为重点内容补充“为什么推荐”
4. 对适合长期沉淀的主题做专题化整理

如果你第一次来，建议先从 [AI Think Brief](/brief/) 和 [Claude Code 专题](/sources/claude-code) 开始。
