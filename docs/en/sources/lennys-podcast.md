# Simon Willison on Agentic Engineering (Lenny Podcast)

> Source: [Highlights from my conversation about agentic engineering on Lenny’s Podcast](https://simonwillison.net/2026/Apr/2/lennys-podcast/)

## Why it matters

This is a high-signal field note on how coding-agent workflows changed software work after late 2025 model upgrades.
The core message is not “AI replaces engineers”, but “the bottleneck moved from implementation to evaluation, testing, and judgment”.

## Key takeaways

1. November 2025 was a practical inflection point for coding reliability.
2. Engineers are a leading indicator for broader knowledge-work disruption.
3. Fast prototyping is now cheap; selection and validation are the hard part.
4. Mid-career engineers face the sharpest transition pressure.
5. Tool fluency is not easy; effective use still requires serious practice.
6. Security workflows need strict verification despite AI-generated report quality.
