# Curation Rules

## Principles

- Signal first: prioritize primary research and measured evidence.
- Verifiability first: every key claim should be traceable to source material.
- Long-term value first: avoid short-lived traffic noise.

## Scoring model

Score each item from 1 to 5 on each dimension (total 15):

- Information density: does it add real knowledge?
- Credibility: does it include data, papers, or open-source proof?
- Actionability: can it be turned into an experiment, project, or decision?

## Update cadence

1. Weekly: add new sources and remove dead links.
2. Monthly: reorder priorities and refresh the roadmap.
3. Quarterly: add a focused page for emerging themes (for example multimodal agents or reasoning optimization).
