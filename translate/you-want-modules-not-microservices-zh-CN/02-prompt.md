# Translation Prompt Context

## Task

Translate Ted Neward's article "You Want Modules, Not Microservices" into polished Simplified Chinese for general readers.

## Requirements

- Preserve the article's section structure and argumentative flow.
- Keep the sarcasm and rhetorical punch where appropriate.
- Translate technical terms accurately, with first-use notes only when helpful.
- Keep quoted slogans and lists easy to scan.
- Do not flatten the distinction between modularity and distributed systems.

## Glossary

- module -> 模块
- microservice -> 微服务
- hype -> 热潮 / 炒作
- pipes-and-filters -> 管道-过滤器架构
- SOA -> 面向服务架构
- Fallacies of Distributed Computing -> 分布式计算谬误
- Fallacies of Enterprise Computing -> 企业计算谬误
- dotted-line reporting -> 虚线汇报关系
- backplane -> 架构底座

## Style notes

- Read like a strong Chinese technical essay.
- Preserve short rhetorical turns such as "… modules."
- Avoid word-for-word translation when the point is rhetorical rather than literal.
